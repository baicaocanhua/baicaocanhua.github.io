多列表拖拽

主要参数是 group

group{
name:”words”,name要相同才能互相拖拽
pull:true, 则可以拖拽到其他列表 否则反之
put:true, 则可以从其他列表中放数据到改列表，false则反之
}
pull: ‘clone’, 还有一个作用是克隆，就是当这个列表拖拽到其他列表的时候不会删除改列表的节点。

a和b可以相互拖拽,group的name要相同才能互相拖拽
